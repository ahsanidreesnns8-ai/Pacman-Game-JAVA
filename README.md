# 🎮 Pacman Game (Java)

This is a simple console-based Pacman game developed using Java.

## 🚀 Features
- Move Pacman using W, A, S, D keys
- Random ghost movement
- Score increases when Pacman moves on dots
- Difficulty levels (Easy, Medium, Hard)
- Game over when ghost catches Pacman

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- 2D Arrays
- Loops and Conditionals
- Random class in Java

## ▶️ How to Run

1. Compile the program:
   javac Pacman_Game.java

2. Run the program:
   java Pacman_Game

## 🎯 Controls

| Key | Action |
|-----|--------|
| W   | Move Up |
| S   | Move Down |
| A   | Move Left |
| D   | Move Right |
| Q   | Quit Game |

## 📌 Project Structure
- Game_Board → Handles board
- Pacman → Player logic
- Ghost → Enemy logic
- Pacman_Game → Main game loop
